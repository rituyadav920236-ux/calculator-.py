# Simple Python Calculator
ye mere ek basic Python project hai jo maine banaya hai .

## Features:
Do numbers ko "Add (+)"  Karna .
Numbers ko "Subtract (-)" karna.
Numbers ko "Multiply (*)" aur "Divide (/)" karna.

## Kasie Chalayein ?
Is repository ko download karein.
Apne computer mein Python install karein.
Terminl mein 'python calculator.py' likh kar run karein.
